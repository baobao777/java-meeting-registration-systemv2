> 版本：v1.0 | 日期：2026-05-21

## 1. 页面结构总览

```
登录页 ─→ 注册页
   │
   ▼
会议列表页 ─→ 会议详情页 ─→ 报名/取消
   │
   ├──→ 我的报名页
   │
   └──→ 管理后台 ─→ 创建/编辑会议页
                   └→ 报名列表（弹窗模态框）
```

## 3. 交互说明

| 操作 | 交互行为 |
|------|---------|
| 点击会议卡片 | 跳转会议详情页 |
| 点击立即报名 | 调用报名API，按钮变为"取消报名" |
| 点击取消报名 | 弹出确认框 → 调用取消API |
| 管理后台删除 | 弹出确认框 → 调用删除API |
| 管理员查看报名 | 弹窗模态框显示报名人员列表 |
| 未登录访问受保护页面 | 自动跳转登录页 |

## 4. 颜色方案

- **主色调**: #1a73e8（蓝色）
- **背景**: #f0f2f5（浅灰）
- **卡片背景**: #ffffff（白色）
- **状态-即将开始**: #e3f2fd 蓝底  #1565c0 蓝字
- **状态-进行中**: #e8f5e9 绿底  #2e7d32 绿字
- **状态-已结束**: #f5f5f5 灰底  #757575 灰字
- **状态-已取消**: #ffebee 红底  #c62828 红字
