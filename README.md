# Java会议报名管理系统

## 项目简介
这是一个基于 Spring Boot 的会议报名管理系统课程项目，实现了用户注册登录、会议创建、报名审核等核心功能。

## 项目结构
- `01-需求分析文档`：项目需求说明与分析报告
- `02-原型设计`：系统交互原型图
- `03-源代码`：后端 Spring Boot 核心代码
- `04-接口测试`：API 接口测试文档
- `05-测试报告`：系统功能测试与性能测试报告
- `06-开发总结报告`：项目开发过程总结
- `07-数据库脚本`：MySQL 建表与初始化脚本
- `08-部署说明文档`：项目本地部署与运行指南
- `11-部署包/deploy`：项目可执行部署包
- `12-会议报名管理系统ppt`：项目答辩PPT
